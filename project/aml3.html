Exploring Fashion MNIST: Clustering, Visualization, Neural Network Training, and Performance Evaluation

Project Information

  • Category: Machine Learning
  • Project date: 31 Mar, 2023
  • Project URL: https://github.com/git-avinashpawar/AML

Technologies used : Python, Jupyter Notebook, Linux, Shell, Git, Github

View on Github

In the "Fashion MNIST Insights" project, I delved into the Fashion MNIST dataset to gain valuable insights and perform various data analysis tasks. The project involved clustering analysis, dimensionality reduction using PCA, training a Random Forest classifier, visualizing data in reduced dimensions, and building a feedforward neural network using Keras. Additionally, I compared different learning rates and activation functions to evaluate their impact on model performance and accuracy.

The following steps were undertaken to explore and analyze the dataset:

  • Dataset Preparation:
    To begin, I downloaded and loaded the Fashion MNIST dataset, which consists of 70,000 grayscale images of 10 different clothing categories. I split the dataset into training (50,000 images), validation (10,000 images), and test (10,000 images) sets, maintaining the same proportions as in previous assignments.
  • PCA and Random Forest Classifier:
    In this step, I applied Principal Component Analysis (PCA) to the training dataset to reduce dimensionality. I determined the number of components required to preserve 95% of the variance. By using the reduced dataset, I trained a Random Forest classifier and evaluated its performance on the test set. A key aspect of analysis involved comparing the training time with that of previous assignments.
  • Visualization:
    To visualize the impact of PCA, I randomly selected and plotted 10 images in their original form. Subsequently, I plotted the same set of images after retaining 95% of the variance using PCA. This allowed for a visual comparison of the images before and after dimensionality reduction. Additionally, I computed the variance explained by the first two principal components.
  • Dimensionality Reduction Visualization:
    Using PCA, t-SNE, LLE, and MDS (Multi-Dimensional Scaling) techniques, I reduced the dimensionality of the dataset to two dimensions. For each method, I plotted 1000 random images from the training set in a scatterplot, using 10 different colors to represent the target class of each image. Finally, I expressed my preference for a particular visualization technique and explained the reasons behind it.
  • Clustering with K-Means:
    Taking 10,000 samples from the training portion of the Fashion MNIST dataset, I applied K-Means clustering. To expedite the process, I employed PCA for dimensionality reduction. To determine an appropriate number of clusters, I utilized one of the techniques covered in class. Finally, I visualized the clusters, showcasing a subset of images within each cluster and inspecting for similarity in clothing items.
  • Clustering with Gaussian Mixture Model (GMM):
    Similarly, I selected 10,000 samples from the training set and performed clustering using a Gaussian Mixture Model. Employing PCA for dimensionality reduction, I ensured an optimal number of clusters based on established techniques. Visualizing the clusters, I examined the similarity of clothing items within each cluster. Furthermore, I generated 20 new clothing items using the GMM model and visualized them by employing the inverse_transform() method.
  • Building a Feedforward Neural Network:
    Using Keras within TensorFlow, I constructed a fully connected (dense) feedforward neural network with two hidden layers. The first hidden layer contained 200 neurons, while the second hidden layer contained 50 neurons. Both hidden layers employed the ReLU activation function. The network was trained for 100 epochs, and I plotted the training and validation loss and accuracy for three different learning rates.
  • Model Evaluation and Analysis:
    For the best-performing learning rate, I evaluated the network's performance on the test portion of the Fashion MNIST dataset, reporting the loss and accuracy metrics. Additionally, I determined the total number of parameters in the network, differentiating between regular parameters and bias parameters. This analysis provided insights into the network's complexity and the impact of bias on its performance.
  • Hidden Layers with Linear Activation Functions:
    To explore the impact of activation functions on accuracy, I repeated the previous steps but used linear activation functions for the hidden layers. This allowed for a comparison of the network's performance with the ReLU activation functions. Finally, I discussed the observed changes in accuracy and provided an explanation for the effects of linear activation functions.

The "Fashion MNIST Insights" project encompassed a comprehensive analysis of the Fashion MNIST dataset. Through dimensionality reduction, clustering, visualization, and neural network training, I gained valuable insights into the dataset's characteristics and explored various data analysis techniques. By comparing different approaches, such as PCA, t-SNE, LLE, MDS, and different activation functions in neural networks, I enhanced my understanding of the strengths and limitations of each method. This project demonstrated my proficiency in data preprocessing, visualization, modeling, and evaluation, showcasing my skills as a data science graduate and providing valuable contributions to the field of fashion image analysis.
