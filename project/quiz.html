COVID-19 Awarness Quiz

Project Information

  • Category: Web design, Volunteer
  • Project date: 24 April, 2020
  • Project URL: https://avinashpawar.dev/Quiz/

Technologies used : HTML5, CSS3, Bootstrap5, Firebase, Git, Github

View a Demo

Introducing an impactful project that was part of the COVID-19 awareness program at Shivaji University, Kolhapur. As a Computer Science student and NSS Volunteer, I developed a highly engaging and educational tool called the COVID-19 Awareness Quiz, which received an astounding response of over 1 million unique participants within just one week. It is a project that embodies the collaboration between data science and social responsibility.

This project was built using a combination of HTML, CSS, JavaScript, and Firebase, leveraging the power of these technologies to create an interactive and informative quiz. As a vital component of Shivaji University's awareness program, the quiz played a crucial role in disseminating knowledge and raising awareness about COVID-19 among students, faculty, and the wider community.

Technical Specifications:

  • HTML: HTML formed the foundation of the COVID-19 Awareness Quiz, enabling the creation of a structured and intuitive user interface. It facilitated the presentation of quiz questions, options, and other essential elements in a user-friendly manner.
  • CSS: CSS played a crucial role in enhancing the visual appeal and overall user experience of the quiz. By applying custom styles and designs, CSS made the quiz interface visually engaging, aligning it with the branding and aesthetic of the awareness program.
  • JavaScript: JavaScript added interactivity and dynamic functionality to the COVID-19 Awareness Quiz. It enabled real-time validation of answers, progress tracking, and score calculation, providing participants with immediate feedback and an immersive quiz experience.
  • Firebase: The integration of Firebase into the quiz ensured seamless user authentication, secure data storage, and real-time updates. Leveraging this powerful platform, the quiz efficiently managed and stored participants' responses, offering a reliable and scalable solution for data storage and retrieval.

As part of Shivaji University's awareness program, the COVID-19 Awareness Quiz garnered an astounding response of over 1 million unique participants within a week. This achievement is a testament to the importance and impact of raising awareness about COVID-19 within the university community.

Upon completion of the COVID-19 Awareness Quiz, participants receives a certificate, recognizing their commitment to increasing knowledge and promoting responsible behavior during the pandemic.

Snapshots from Project
