Predicting Insurance Data: Exploring Regression Models for Accurate Predictions and Analysis

Project Information

  • Category: Machine Learning
  • Project date: 10 Feb, 2023
  • Project URL: https://github.com/git-avinashpawar/AML

Technologies used : Python, Jupyter Notebook, Linux, Shell, Git, Github

View on Github

The "Predicting Insurance Data" project focuses on exploring insurance data prediction through the application of Linear Regression and Polynomial Regression models. The project incorporates regularization and validation analysis to enhance the accuracy and reliability of the predictions.

The project includes various steps and analyses to gain insights into the insurance data:

  • Data Summarization: The project starts by summarizing the available data, including the assessment of data size and identifying continuous valued and categorical attributes. Special treatment requirements for specific attributes are identified and addressed.
  • Statistical Analysis and Visualization: Statistical values are computed for each attribute, accompanied by visualizations such as histograms to understand attribute distributions. Key attributes are analyzed for notable traits and patterns.
  • Relationship Analysis: The relationships between data attributes and the target label are examined by computing the Pearson Correlation Coefficient (PCC) and generating scatter plots. This analysis helps identify correlations and dependencies within the data.
  • Data Splitting: A representative portion (20%) of the data is randomly selected for testing, ensuring its representativeness and reliability.
  • Model Training and Evaluation: Linear Regression models are trained using the training data, employing four-fold cross-validation and appropriate evaluation metrics. The models are trained using closed-form solutions (Normal Equation or SVD) and Stochastic Gradient Descent (SGD) algorithms. Regularization techniques such as Ridge, Lasso, and Elastic Net are applied with varying penalty terms to study their impact. The project explores the influence of hyperparameters such as batch size and learning rate on the model's performance. Training and validation losses are plotted to track the model's learning progress.
  • Polynomial Regression Analysis: The project repeats the previous step with Polynomial Regression, assessing whether the model overfits or underfits the data. The validation loss is used to evaluate model performance.
  • Prediction and Performance Evaluation: The trained models are utilized to make predictions on the test data, and performance is summarized using appropriate evaluation metrics. The results are discussed, and potential avenues for further improvement are explored.

Throughout the project, a comprehensive analysis of the insurance data is conducted, employing regression models and advanced techniques to predict outcomes accurately. The project provides insights into the data, identifies relationships, and offers recommendations for enhancing prediction performance.
