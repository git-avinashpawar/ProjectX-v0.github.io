Exploring Sign Language MNIST: Classification Analysis with Multinomial Logistic Regression, SVM, and Random Forest

Project Information

  • Category: Machine Learning
  • Project date: 03 Mar, 2023
  • Project URL: https://github.com/git-avinashpawar/AML

Technologies used : Python, Jupyter Notebook, Linux, Shell, Git, Github

View on Github

The project "Sign Language MNIST Exploration" focuses on the analysis of the Sign Language MNIST dataset using various machine learning algorithms. The dataset is used to classify hand gestures representing sign language letters into different categories.

The following steps were undertaken to explore and analyze the dataset:

  • Dataset Download and Loading: The dataset was downloaded and loaded, consisting of pre-split train and test sets. A validation set was created by selecting 10,000 samples from the training set, resulting in a total of 50,000 training samples and 10,000 validation samples. A few sample images were displayed for visualization.
  • Multinomial Logistic Regression: The dataset was trained using the Multinomial Logistic Regression algorithm, also known as softmax regression. Different hyperparameters were optimized to maximize performance. Training, validation, and testing performances were evaluated and compared, discussing the impact of hyperparameters on the results.
  • Support Vector Machines (SVM): Support Vector Machines were employed for classification, considering different kernels to enhance performance. Hyperparameter tuning was performed to optimize the SVM models. Similar to the logistic regression approach, training, validation, and testing performances were analyzed and discussed.
  • Random Forest Classifier: The Random Forest algorithm was utilized for classification, and feature importance analysis was conducted to understand the significance of different features in the classification process. Training, validation, and testing performances were evaluated, and the impact of hyperparameters was discussed.
  • Ensemble Learning: The classifiers from the previous steps were combined into an ensemble model to achieve better performance than individual classifiers on the validation set. The ensemble model was then tested on the separate test set. Findings from the ensemble approach were described and discussed, highlighting the improved performance achieved by combining multiple classifiers.

The project aims to explore the Sign Language MNIST dataset and compare the performance of different machine learning algorithms. By analyzing various classifiers and optimizing hyperparameters, the project aims to identify the most effective approach for sign language gesture classification.
