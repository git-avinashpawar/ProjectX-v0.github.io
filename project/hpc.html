Multithreaded Parallel K-means Clustering

Project Information

  • Category: High-Performance Computing, Data Science
  • Project date: 09 Dec, 2020
  • Project URL: https://github.com/git-avinashpawar/HPC-Parallel-K-means

Technologies used : C++, MPI, OpenMP, Linux, Shell, Git, Github

View on Github

Our project focuses on utilizing High-Performance Computing (HPC) techniques to implement parallel K-means clustering. By leveraging the power of parallel computing, we aim to improve the performance and efficiency of the K-means clustering algorithm. The project is implemented using C++ and utilizes both OpenMP and MPI frameworks.

K-means clustering is a popular unsupervised machine learning algorithm used for grouping data points into clusters based on their similarities. However, the algorithm can be computationally expensive, especially when dealing with large datasets. To address this challenge, our project takes advantage of parallel computing techniques to accelerate the clustering process.

We employ OpenMP, a shared-memory parallelization framework, to leverage multiple threads on a single node of the machine. This allows for efficient utilization of the available computational resources within a single computing unit.

In addition to OpenMP, we utilize the Message Passing Interface (MPI) framework, which enables communication and coordination among multiple nodes of a distributed computing system. By leveraging MPI, we can achieve parallelism across a large number of computing nodes, effectively scaling the computation to handle extensive datasets and complex clustering tasks.

For our project, we specifically target a machine comprising 140 nodesnhaving 64 cores each, enabling highly parallelized processing. The combination of OpenMP and MPI enables us to exploit both shared-memory parallelism within individual nodes and distributed-memory parallelism across multiple nodes.

By implementing parallel K-means clustering using HPC techniques, we aim to achieve faster and more efficient clustering of large datasets. This project will not only demonstrate the benefits of parallel computing but also showcase the effectiveness of combining OpenMP and MPI frameworks for high-performance data analysis tasks.

Overall, our project offers an innovative approach to K-means clustering by harnessing the power of HPC, OpenMP, and MPI. It addresses the challenges associated with large-scale data clustering and provides an efficient solution that can scale to a 1000-node machine, significantly reducing the computation time required for clustering tasks.
